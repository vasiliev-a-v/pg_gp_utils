Тестирование утилиты adbmt (Arenadata DB Magic Tool)

Коллеги, добрый день!
Я написал утилиту (adbmt) для сбора диагностической информации.
При создании вдохновлялся аналогичной утилитой gpmt от наших конкурентов.
gpmt GPMT (Greenplum Magic Tool):
https://techdocs.broadcom.com/us/en/vmware-tanzu/data-solutions/tanzu-greenplum/6/greenplum-database/utility_guide-ref-gpmt.html

Утилита adbmt (Arenadata DB Magic Tool) предоставляет набор инструментов (tool) для сбора диагностической информации, необходимой службе техподдержкой поддержки Arenadata.
Утилита adbmt - это приблизительный аналог утилиты gpmt.

Пока из всех "инструментов" реализован только gp_log_collector.
Но собственно, он больше всего и нужен.

Есть необходимость утилиту поиспытывать.
